This project is a YOLOv8 (You Only Look Once, version 8) object detection system designed to identify and classify e-waste items in images. It has been trained to detect a variety of electronic waste objects such as smartphones, cables, batteries, and other discarded electronic components. The model outputs bounding boxes with class labels and confidence scores, enabling efficient recycling and waste management.
